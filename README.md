I want to create a mobile app for iPhone that helps tracking Pomodoro's during the day.
It should allow to create several categories, that is types of Pomodoro, for example - personal work, etc. And it should allow to start Pomodoro, that is 25 minutes timer, and once the timer rings it should ask which category to assign this Pomodoro to.
At the end of the day it should show Summary how many Pomodoro's done overall and per each type, and it should be able to show a weekly table with these figures.
